# gyandistro-backend-nestjs-functions

Blogging app

## Description

Sample Nest.js, TypeScript starter repository of a blogging app GYANDISTRO.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
